Cascada y Herencia CSS

Cascada

Los estilos se aplican en cascada, es decir si tenemos un mismo estilo definido para una etiqueta o una clase siempre se aplicará el último. 

            
            .contenedor-cascada p {
                color:black;

            }

            .contenedor-cascada p {
                color:skyblue;
            }

Herencia

Exiten elementos HTML que herendan propiedades de sus elementos padres y otros no.

Por ejemplo la propiedad color y font-size son propiedades heredables por los elementos hijos. Margin padding, border y background no.

Hay que tener el claro el concepto de herencia para conocer que estilos se aplican a cada elemento

                
                .contenedor {
                    font-size: 1.2em;
                    color: #ddd;
                }
