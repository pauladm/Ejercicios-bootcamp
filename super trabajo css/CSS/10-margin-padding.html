Margin y padding CSS

Margin

                
                margin: 10px 10px 10px 10px; /*  top  left bottom right */
                margin: 20px 30px; /* top y bottom left y right */
                margin-top: 10px; /* top */
                margin-left: 20px; /* left */
                margin-right: 35px; /* right */
                margin-bottom: 12px; /* bottom */
                margin: 0px; /* todos */

Padding

                
                padding: 10px 10px 10px 10px; /*  top  left bottom right */
                padding: 20px 30px; /* top y bottom left y right */
                padding-top: 10px; /* top */
                padding-left: 20px; /* left */
                padding-right: 35px; /* right */
                padding-bottom: 12px; /* bottom */
                padding: 0px; /* todos */

Margin auto

Estable un margin automático horizonal. Se utiliza para centrar contenidos dentro de un contenedor

                
                    margin:0px auto;
                    margin:10px auto 15px auto;

Ejemplos

                
                .contenedor-cajas {
                    width: 100%;
                    margin:0 auto;
                    text-align: center;
                }

                    width: 100px;
                    /*float: left;*/
                    height: 100px;
                    margin:2px;
                    font-size: 0.8em;
                    display: inline-block;
                }

                .caja1 {
                    background-color: antiquewhite;

                .caja2 {
