Sintaxis CSS

  • Selector

    Permite seleccionar los elementos del html que van a ser modificados

  • Declaración

    Conformada por una serie de propiedades (font-size, font-family, etc...) y por cada propiedad le asignaremos un valor después de : Acabaremos la definición con ;

Ejemplo

                
                p {
                    font-size: 2em;
                    font-family: Arial, Helvetica;
                }

Se aplicará el cambio a todos los <p> de nuestro documento HTML
