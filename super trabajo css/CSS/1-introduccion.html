Introducción a CSS

¿Qué es CSS?

Lenguaje de hojas de estilos para controlar la presentación de documentos de lenguaje de marcados como el HTML.

Con HTML construiremos la estructura de nuestro sitio web y con CSS los acabados de estilos (capa de presentación que se puede cambiar sin tocar la estructura.

Las hojas de estilos son una colección de reglas de formato, están contenidos generalmente en archivos externos con extensión .css aunque es posible trabajar con estilos incluidos directamente en archivos HTML o a través de la etiqueta style.

Podremos crear una hoja de estilo css que se le pueda aplicar a todos nuestros archivos HTML de nuestro sitio web, haciendo el mantenimiento muy fácil y rápido. Imagínense que quieren cambiar el tamaño de fuente de todos los encabezados de mi sitio web. Con una sola línea de código 

                
                    h1 {
                        font-size: 2em;
                    }

Estilos por defecto de los navegadores

Por defecto cada navegador posee una hoja de estilos, es por eso que cuando abrimos una página web que no tiene ninguna hoja de estilo definida, se ve de la forma que se ve. Los títulos tienen un tamaño en concreto, los vínculos también. Existen unas reglas css implícitas por el navegador que son las que se utilizan. Cuando aplicamos nuestros CSS lo que hacemos es sobreescribir los que ya existen por defecto.
