Tipos de Selectores CSS

  • Selector de tipo elemento

                        
            p {
                color:red;
            }

  • Selector de tipo clases

                        
            .color-verde {
                color:green;
            }

  • Selector de tipo id

                        
            #sidebar {
                width:30%;
            }

  • Selectores específicos

                        
            h2.color-azul {
                color:blue;
            }

  • Selectores de tipo descendentes o selectores compuestos

                        
            div p span {
                font-size:20px;
                color:#ccc;
                text-decoration: underline;
            }

  • Selectores agrupados

                        
            h4, h5 {
                font-size:18px;
                color:#bbb;
                font-weight: normal;
            }
