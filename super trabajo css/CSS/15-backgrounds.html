Background CSS

Establece el fondo de elementos html

            
            background-color: #ccc; 
            background-image: url(ruta de la imagen);
            background-repeat: repeat | repeat-x | repeat-y | no-repeat | inherit
            background-position: horizontal vertical (Porcentaje | valor | left | center | right | top | center bottom)
            background-size: auto | length | cover| contain | initial| inherit;
            background: #FFF url(ruta) no-repeat left top

Ejemplos

            
            .contenedor1 {
                border:2px solid red;
                padding:10px 5px 10px 5px;
                margin: 15px 10px 15px 10px;
                width:120px;
                height: 120px;
                background-color: red; /* Establecemos el color de fondo */
                color:#fff;
            }

            .contenedor1 p {
                font-size: 0.8em;
                text-align: justify;
                background-color: yellow; /*color de fondo del párrafo*/
                color:#000;
                padding:2px;
            }
