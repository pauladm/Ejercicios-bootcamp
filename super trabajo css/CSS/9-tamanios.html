Tamaños CSS

  • width

    ancho del elemento

  • height

    Alto del elemento

Unidades de medida

  • px

    Pixeles. Fijo

  • %

    Porcentaje. Heredable

  • em

    Tamaño por defecto de la fuente según el dispositivo (16px desktop). Heredable

  • rem

    Mismo valor que em pero no heredable
