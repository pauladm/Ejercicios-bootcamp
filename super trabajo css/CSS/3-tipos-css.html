Tipos de CSS

  1. Inline

    Atributo style de una etiqueta html

                            
            <p style="color:rgb(255, 0, 0)">Color rojo</p>

    Color rojo

  2. Interno

    Embebido en el propio html

                            
            <style>
                #ejemplo p {
                    color:greenyellow;
                }
            </style>

    Corlor verde

  3. Fichero externo

                            
            <link href="css/main.css" rel="stylesheet" type="text/css">

    Otro color
