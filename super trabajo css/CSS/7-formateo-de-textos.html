Formateo de Textos CSS

  • font-family

    Tipo de fuente 
                        
            p {
                font-family:Arial;
            }

  • font-size

    Tamaño de la fuente 
                        
            p {
                font-size:1.4em; /* Unidades de tamaño en %,px,em,rem*/
            }

  • font-weight

    Peso de la fuente 
                        
            p {
                font-weight:bold; /* normal, light, bold o número especifico*/
            }

  • font-style

    Estilo de la fuente 
                        
            p {
                font-style:normal; /* normal, italic, oblique, inherit */
            }

  • font-variant

    Todas las letras minúsculas se convierten a letras mayúsculas. Sin embargo, las letras mayúsculas convertidas aparecen en un tamaño de letra más pequeño que las letras mayúsculas originales en el text 
                        
            p {
                font-variant: small-caps;
            }

  • text-transform

    Transforma la fuente 
                        
            p {
                text-transform: uppercase;
            }

  • text-align

    Alineación del texto 
                        
            p {
                text-align: center;
            }

  • text-indent

    Desplaza la primera línea 
                        
            p {
                text-indent: 1em;
            }

  • letter-spacing

    Espacio entre caracteres 
                        
            p {
                letter-spacing: 1em;
            }

  • line-height

    Espacio entre líneas 
                        
            p {
                line-height: 1em;
            }

  • color

    color del texto (Código Hexadecimal) 
                        
            p {
                color: #cccccc;
            }

Ejercicio de investigación

Averigua todo lo que puedas sobre la propiedad font: y sus distintos valores posibles

            p.ejemplo {
                font-family: arial;
                font-size: 1em; 
                font-weight: bold; 
                font-style: normal;
                text-transform: uppercase; 
                text-align: left;
                text-indent: 1em; 
                letter-spacing: 0.1em; 
                line-height: 1.2em; 
                color: #cccccc; 
            }
